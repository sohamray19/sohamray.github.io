  • Machine Learning (Industry Experience)

    I am deeply interested in machine learning algorithms, specifically NLP. I am also interested in scaling these algorithms to fit large datasets.
    This is what I've done so far:

    • Cybage Software: At Cybage, I was a part of the Decision Mines team, the machine learning division. Here, I built a feedback analysis system that can analyse sentiment of text, irrespective of domain.

      It proved especially challenging to abstract the domain, since most text classification tasks are heavily dependent on context. To tackle this problem, I combined the Yelp, Amazon and Hotel Reviews datasets available online and purged the domain dependent words among them. Next, I wrote a data preprocessing pipeline to clean, preprocess and enforce labelling on the data. However, this took tremendously long to preprocess, so I used python multiprocessing to speed up the process. Finally, for sentiment analysis, I used a deep learning model, with word embeddings, and LSTMs. For hyperparameter tuning, I used Bayesian Optimization to achieve an optimal result, in minimal time.

      Our final model improved classification accuracy by 11% on the inplace system, and scored an accuracy of 80% on data from un-trained domains, that is book data, kitchen appliance review data and banking data.

      For this project, I worked with python and used the Tensorflow, NLTK and scipy libraries.

    • ASquared IoT: Here, I'll focus on the machine learning backend of the web-app I built at ASquared IoT. To perform sound classification in a factory effectively, I built a denoising model to extract the sound of interest, and a classification model to label the weld as satisfactory or defective.

      So far, theres been limited research on sound classification, especially for industrial sounds. To tackle this problem, and to convert our sound classification problem into an image classification problem, I converted the sound signals to spectrograms (a visual representation of sound). To analyse these images, I used a convolutional neural network, achieving an accuracy of 71%. Unfortunately, this wasn't accurate enough for testing real welded goods. When I delved deeper, I found the reason to be noise in the input sound. To mitigate this, I constructed denoising models using statistical (fourier transforms, spectral subtraction, etc.) and ML methods. Another deep learning approach, deep denoising auto encoders, provided the best results and boosted projected classification accuracy to 84%.

      Drawback: A drawback of the final model was little variance in the input data. Unfortunately, all data was collected by our team in person, with help from field engineers in a factory we partnered with, restricting its magnitude and diversity.

      For this project, I worked with python and used the Tensorflow, scipy and librosa libraries.
